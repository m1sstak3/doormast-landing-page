# DoorMast — Лендинг сервиса по установке дверей

Профессиональный одностраничный веб-сайт (Landing Page) для сервиса по быстрой установке межкомнатных и входных дверей в Москве. Разработан с упором на высокую конверсию, современный дизайн (Glassmorphism) и надежную кодовую базу.

## 🎯 Цель проекта
Обеспечение стабильного потока целевых заявок и звонков. Проект решает проблему клиента: быстрый вызов замерщика, прозрачный расчет стоимости "онлайн" и уверенность в гарантии качества.

## 💻 Стек технологий
- **HTML5:** Семантическая верстка, обеспечение доступности.
- **CSS3 (Vanilla):** Использование CSS Variables для архитектуры z-index слоя и цветовой палитры, Flexbox/Grid layouts, современные визуальные эффекты (backdrop-filter). Адаптивный mobile-first дизайн.
- **JavaScript (Vanilla JS):** Плавная прокрутка, логика многошагового калькулятора, маска телефонного номера (без сторонних библиотек), перехват и валидация форм, защита от спама (Honeypot).

## 🏗 Архитектура проекта
Проект организован с четким разделением ответственности (Separation of Concerns):
```
/
├── assets/
│   ├── css/
│   │   └── style.css       # Глобальные стили, переменные, UI-кит
│   ├── js/
│   │   └── script.js       # Логика калькулятора, UI-взаимодействия
│   └── images/
│       └── hero.png        # Оптимизированные изображения
├── index.html              # Главная бизнес-логика (включая data-* атрибуты)
└── README.md               # Документация проекта
```

## ✨ Основной функционал
- 🧮 **Динамический онлайн-калькулятор:** Учитывает тип двери (получает цену из `data-price`), количество и галочку "срочности", мгновенно пересчитывая итог.
- 📱 **Мобильная адаптация:** Полноценное выдвижное меню и кнопка быстрого звонка для мобильных пользователей.
- 🛡️ **Безопасность и спам-защита:** Интегрированное поле `bot_field` (Honeypot) блокирует автоматические бот-отправки без ущерба для UX.
- 🚀 **Оптимизация:** Ленивая загрузка изображений (`loading="lazy"`), минимизация repaint'ов, отказ от тяжеловесных библиотек вроде jQuery или Bootstrap.
- 📞 **Smart Format:** Автоматическая маска при вводе телефона `+7 (999) 000-00-00`.

## 🚀 Инструкция по запуску
Так как проект написан на чистом HTML/CSS/JS, для его запуска не требуется установка зависимостей.
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/doormast-landing.git
   ```
2. Откройте файл `index.html` в любом современном браузере.
3. *Для разработки:* Рекомендуется использовать расширение Live Server (VS Code) для автоматической перезагрузки страниц.
